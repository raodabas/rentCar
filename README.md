#Car Rental App

This project a full-stack Car Rental System built using Node.js, Express.js, MongoDB, and AngularJS.
It allows users to browse available vehicles, view details, select date ranges, and rent cars though an intuitive and responsive interface.

User Authentication:
-User registration
-Secure system
-MongoDB integration for storing user accounts

Vehicle Management:
-Browse available vehicles
-View detailed vehicle information
-Filter cars by availability based on selected date rangers

Rental Functionality:
-Select start and end dates
-Check availability instantly
-Submit a vehicle rental rewuest
-store rental data in MongoDB
